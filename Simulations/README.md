This folder contains the simulation-related work for the 4-axis robotic arm project.

The goal of this phase is to validate the robot’s structure(URDF), joint behavior, motion workflow & path planning in a virtual environment. The simulations are being developed using ROS2, RViz, and Gazebo.

## Purpose

The simulation workflow helps in:
- Verifying robot kinematics and joint orientation
- Understanding workspace and motion behavior
- Testing robot movement before hardware development
- Building familiarity with robotics software tools and workflows

## Tools Used

- ROS2 Jazzy
- RViz2
- Gazebo
- [SolidWorks URDF Exporter](https://wiki.ros.org/sw_urdf_exporter)
- Ubuntu Linux 24.04
